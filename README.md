# img_conv

画像ファイルを変換します。

## complementary_color

画像を補色に変換します。

入力す画像のパスと出力する画像のパスを入力します。

## 注意

PNGの透明部分は黒くなります。

```opencv```が必要です。

```bash
# pip3 install
sudo apt update
sudo apt install python3-pip

# opencv install
pip install opencv-python
pip install opencv-contrib-python
```
